My portfolio

These are some of the projects that I have done that represent my interests, skills, and aspirations.

Atlas News

A website built using HTML, CSS, JavaScript, SQLAlchemy, and Flask for the Bay Area Hacks Hackathon. Atlas News provides a unique user interface to access international news using the newscatherAPI. It offers the ability to create an account/login to leave comments on any article, and has an interactive map UI to find relevant news in specific regions of the world.

Cryptometer

CryptoMeter is a web-app that is built with Html, Css, Python, and Javascript. Uses the Twitter API and Huggingface Roberta model to calculate the sentiment of Bitcoin and Ethereum to predict their respective prices.

Know Your Bias

A web-app that allows the user to input any news article and returns the political bias of the article. The website also gives the user articles that have the opposite political bias.

Signify

Signify is a practical tool designed to convert American Sign Language (ASL) gestures into English text. Built using TensorFlow, Python, and JavaScript. Users can also choose to have the translated text read aloud, facilitating communication between the deaf and hearing communities.
